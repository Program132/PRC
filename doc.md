## Started

* How to print something ?

    `print YOUR ARGUMENT`



* How declare a variable ?

    `variable NAME = VALUE`



* How to set an already existing variable ?

    `set NAME NEW-VALUE`



* How to see a value of a variable ?

    `send NAME`



## Files

* How to read my file ?

    `files read myFile.txt`

* How to write in my file ?
